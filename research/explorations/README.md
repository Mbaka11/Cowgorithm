# Adjacent Market Explorations

This directory contains market explorations generated by the OpenClaw research agent. Each file covers one market vertical outside the core four segments (cattle, livestock, pets, sport), scored using the 5-factor opportunity framework.

## Scoring Framework

| Factor | Weight |
|--------|--------|
| Market size (QC/Canada) | 25% |
| Competitive gap | 25% |
| Technology transfer difficulty | 20% |
| Regulatory fit | 15% |
| Strategic alignment with Cowgorithm | 15% |

Opportunities scoring **≥ 3.5/5** are flagged for deeper analysis.
